Instructions on using ManageYourTask Application

  1. Add button (+) in the toolbar makes you to create new tasks
  2. Tasks are placed in the priority order and Tasks with the same priority order are arranged alphabetically
  3. Each task has Edit and Delete buttons if you want to change the details or remove the task
  4. Setting in the toolbar adjusts the length and day of timer
  5. Pending Tasks in this app : Adding a Timer to the tasks are next steps in the app development

ManageYourTask Application

This application lets users create and manage their tasks and also reminds them of when they have scheduled it. It also lets users allocate their time for each task and smartly reminds them if it overlaps with another task they have created.

Features of ManageYourTask Application

  • Any number of tasks
  • Tasks with goals
  • Tasks with indefinite goals (no goal)
  • Task descriptions
  • Start, pause, reset, delete, edit tasks
  • Progress bars
  • Prioritize tasks based on the order
  • Edit task names, time spent, and goal-times
  • Desktop notifications
  • Sound notifications (default and custom)
